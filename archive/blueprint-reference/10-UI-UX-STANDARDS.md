# 10 — UI/UX և դիզայնի հիմքային կանոններ

## Ընդհանուր ուղղություն
- պրեմիում, մաքուր, հանգիստ, վստահելի տեսք
- ոչ խճճված ինտերֆեյս
- հիմնական գործողությունները առաջին պլանում
- վտանգավոր գործողությունները՝ հստակ հաստատմամբ

## Վերօգտագործվող UI բաղադրիչներ
- աղյուսակ
- քարտ
- ժամանակագիծ
- կարգավիճակի նշան
- մոդալ պատուհան
- քայլային ձև
- դատարկ վիճակ
- վերբեռնման դաշտ
- հաստատման պատուհան
- տպման դիտում

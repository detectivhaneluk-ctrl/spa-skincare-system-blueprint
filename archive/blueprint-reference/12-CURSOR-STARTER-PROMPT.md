# 12 — Cursor մեկնարկային հրահանգ

Այս նախագիծը SPA & Skincare պրեմիում համակարգ է։
Դու պետք է աշխատես ոչ թե որպես պատահական կոդ գրող, այլ որպես համակարգային ճարտարապետ և մաքուր կոդով ծրագրավորող։

## Քո պարտադիր կանոնները
1. Պահպանիր մոդուլային կառուցվածքը։
2. Մի խառնիր համակարգային միջուկը UI-ի հետ։
3. Ամեն նոր ֆայլը դիր ճիշտ պանակի մեջ։
4. Իրավունքների ստուգումը պահիր policy կամ middleware շերտում։
5. Բիզնես տրամաբանությունը պահիր service շերտում։
6. Տվյալների կարդալ/գրելը պահիր repository շերտում։
7. Request validation-ը պահիր առանձին շերտում։
8. Audit log-ը հաշվի առ կարևոր գործողությունների համար։
9. Status փոփոխությունները մի արա պատահական if-երով, պահիր կենտրոնացված կանոններով։
10. Կարգավորելի արժեքները մի թաղիր կոդի մեջ։

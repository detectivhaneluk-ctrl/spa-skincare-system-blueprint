# 05 — Համակարգային միջուկ

Համակարգային միջուկը user-ը չի տեսնում, բայց ամբողջ համակարգը սրա վրա է աշխատում։

## Կենտրոնական ենթահամակարգեր
- կարգավիճակների շարժիչ
- բիզնես հոսքերի շարժիչ
- գործողությունների պատմություն
- ժամանակագծի շարժիչ
- ծանուցումների հերթ
- ֆայլերի շարժիչ
- կոդերի գեներատոր
- փափուկ ջնջում և վերականգնում
- ընդհանուր որոնում
- բախումների շարժիչ
- գնագոյացման շարժիչ
- հարկային շարժիչ
- իրավունքների շարժիչ
- մասնաճյուղային շերտ
- պահուստավորում և վերականգնում

## Ոսկե կանոն
Միջուկը պետք է լինի առանձին և չպետք է լուծարվի առանձին մոդուլների ներսում։

# Փաստաթղթեր և համաձայնություններ

## Նպատակ
Սա փաստաթղթեր և համաձայնություններ մոդուլի նկարագրական փաստաթուղթն է։

## Ենթաբաժիններ
- գլխավոր էջ
- ցանկ / ցուցակ
- դիտում
- ստեղծում
- խմբագրում
- կարգավիճակներ
- գործողությունների պատմություն

## Կախվածություններ
Սա պետք է կառուցվի ընդհանուր manifest-ի և build order-ի համաձայն։

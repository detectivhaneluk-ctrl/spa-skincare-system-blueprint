# Նույնականացում և անվտանգություն

## Նպատակ
Սա նույնականացում և անվտանգություն մոդուլի նկարագրական փաստաթուղթն է։

## Ենթաբաժիններ
- գլխավոր էջ
- ցանկ / ցուցակ
- դիտում
- ստեղծում
- խմբագրում
- կարգավիճակներ
- գործողությունների պատմություն

## Կախվածություններ

# Վաճառք, վճարումներ և դրամարկղ

## Նպատակ
Սա վաճառք, վճարումներ և դրամարկղ մոդուլի նկարագրական փաստաթուղթն է։

## Ենթաբաժիններ
- գլխավոր էջ
- ցանկ / ցուցակ
- դիտում
- ստեղծում
- խմբագրում
- կարգավիճակներ
- գործողությունների պատմություն

## Կախվածություններ
Սա պետք է կառուցվի ընդհանուր manifest-ի և build order-ի համաձայն։

# Օրացույց և ամրագրումներ

## Նպատակ
Սա օրացույց և ամրագրումներ մոդուլի նկարագրական փաստաթուղթն է։

## Ենթաբաժիններ
- գլխավոր էջ
- ցանկ / ցուցակ
- դիտում
- ստեղծում
- խմբագրում
- կարգավիճակներ
- գործողությունների պատմություն

## Կախվածություններ
